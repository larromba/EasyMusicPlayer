==== Name ====

Easy Music Player

==== Description ====

«Easy Music Player» - это простой и альтернативный способ воспроизведения музыки, сохраненной на вашем телефоне. Это приложение было с любовью создано только для одной цели - хорошо играть свою музыку.

Содержит:
- Воспроизвести, Пауза, Следующая, Предыдущая, Перемешать, Поиск
- Тайм-скруббер
- Повторить нет / один / все
- Воспроизведение музыки в фоновом режиме
- Управляйте музыкой с рабочего стола и экрана блокировки

Совместим с iPad и iPhone. Оптимизировано для iPhone 8.
